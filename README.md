# Haatzmaut

מערכת ניהול מרפאה פנימית בשפה העברית (RTL) עבור:
- מטריצת תפוסת חדרים שבועית
- טעינת לו"ז בסיס מקבצי CSV/JSON
- בקשות שינוי חד-פעמיות עם אישור/דחייה וסימון ויזואלי
- ניהול משתמשים ותפקידים לפי צוותים
- מרחב אדמין לעריכת לוח הזמנים, מחיקת/עדכון משבצות ועריכת חדרים
- תצוגת שבוע עבודה מיום ראשון עד חמישי, בין 08:00 ל-20:00, במקטעי 30 דקות
- מודול ישיבות צוות שבועיות (יום ראשון) עם העלאת קבצים
- מרכז שיתוף ידע ומשאבים
- סינון חדרים לפי התאמה קלינית
- התראות פנימיות/ארגוניות
- דיווח תקלות חדרים והעברתם לתור אדמיניסטרציה
- כניסה מאובטחת ושמירה על גישה למסכים פנימיים למשתמשים מחוברים בלבד

## הפעלה מקומית

```bash
cd /workspaces/Haatzmaut
python -m http.server 8000
```

ואז לפתוח בדפדפן: `http://localhost:8000`

## משתמשי הדגמה
- `admin` / `admin123`
- `staff` / `staff123`

## הערות שימוש
- משתמש `admin` רואה את מרחב האדמין ויכול לערוך חדרים, להוסיף או למחוק משבצות בלו"ז ולנהל משתמשים.
- משתמש `staff` יכול לצפות בלוח, לשלוח בקשות שינוי ולדווח על תקלות.
